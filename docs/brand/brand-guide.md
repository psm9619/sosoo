## 1. 브랜드 철학 (Brand Philosophy)

### Mission Statement

> "스피치 연습의 막막함을 해결하고, 누구나 자신감 있게 말할 수 있도록 돕는다."

### Vision

세계에서 가장 개인화된 AI 스피치 코칭 플랫폼이 되어, 면접·발표·소통의 장벽을 낮춘다.

### Core Values

| 가치 | 설명 | 서비스 반영 |
|------|------|------------|
| **Personalization (개인화)** | 내 상황, 내 문서, 내 성장 패턴 기반 코칭 | Progressive Context, 맞춤 질문 생성 |
| **Empathy (공감)** | 남의 목소리가 아닌 내 목소리로 시연 | Voice Cloning |
| **Continuity (연속성)** | 일회성 분석이 아닌, 성장을 기억하는 코칭 | Short-term Memory, 패턴 학습 |
| **Trust (신뢰)** | 투명한 분석과 안전한 데이터 보호 | 명확한 피드백, 프라이버시 정책 |

## 2. 브랜드 포지셔닝

### 타겟 유저: Urgent Performance

**Primary Persona: 김민지 (29세, 마케팅 매니저)**

| 항목 | 설명 |
|------|------|
| **상황** | 3일 후 대기업 이직 면접. "성과 설명" 질문이 걱정 |
| **목표** | 핵심 질문 3개 답변을 자연스럽게 만들고 싶다 |
| **Pain Point** | 객관적 피드백 받을 곳이 없음. 전문 코칭은 비싸고 시간 없음 |
| **핵심 니즈** | 즉각적인 개선, 숙제가 아닌 즉시 적용 가능한 팁 |

**Product Tone:** "장기 코칭 프로그램"이 아닌 **"즉석 컨설팅"**

### 경쟁사 대비 차별점

| 경쟁사 | 제공 가치 | Gap |
|--------|----------|-----|
| **전문 코치** | 고품질 피드백 | $100+/시간, 시간 제약 |
| **Orai, Speeko** | 점수/피드백 | 개선안 시연 없음 |
| **Yoodli** | AI 면접 분석 | Voice Cloning 없음, 컨텍스트 기반 질문 없음 |

### Unique Value Proposition (UVP)

> **"분석 + 시연 + 내 목소리"** 삼박자를 모두 갖춘 유일한 서비스

| Value | Description |
|-------|-------------|
| **Analysis** | "왜 어색한지" 구체적으로 설명 |
| **Demonstration** | "어떻게 고쳐야 하는지" 개선 버전으로 직접 시연 |
| **Empathy** | 남의 목소리가 아닌 **내 목소리**로 개선판을 들려줌 |

## 3. 3대 핵심 차별점

### 1. Voice Cloning

- **경쟁사 현황:** Orai, Yoodli에 없음
- **VoiceUp 제공:** 개선안을 **내 목소리**로 시연
- **핵심 가치:** 남의 완벽한 발음이 아닌, "나도 이렇게 말할 수 있구나" 체감

### 2. Context-Based Practice

- **경쟁사 현황:** 대부분 범용 질문만 제공
- **VoiceUp 제공:** 이력서/발표자료 분석 → 맞춤 질문 생성
- **핵심 가치:** "어떻게 말할지"뿐 아니라 "무엇을 말할지"까지 준비

### 3. Progressive Context

- **경쟁사 현황:** 세션별 독립 분석만
- **VoiceUp 제공:** AI가 유저 성장 패턴 학습 → 개인화 피드백
- **핵심 가치:** 성장 중인 부분 격려, 지속적 약점 우선 피드백

```
┌─────────────────────────────────────────────────────────────┐
│                    Progressive Context                       │
│                                                              │
│  ┌─────────────────────┐    ┌─────────────────────────────┐ │
│  │   Long-term Memory  │    │    Short-term Memory        │ │
│  │   (문서 컨텍스트)    │    │    (최근 3개 세션)          │ │
│  │                     │    │                             │ │
│  │  • 이력서/발표자료   │    │  • 성장 중인 부분 (격려)    │ │
│  │  • 핵심 키워드 추출  │    │  • 지속적 약점 (우선 피드백) │ │
│  │  • 경험/성과 분석    │    │  • 최근 피드백 요약         │ │
│  │  → 맞춤 질문 생성    │    │                             │ │
│  └─────────────────────┘    └─────────────────────────────┘ │
│                    │                     │                   │
│                    └──────────┬──────────┘                   │
│                               ▼                              │
│                    ┌─────────────────────┐                   │
│                    │  개인화된 AI 분석   │                   │
│                    └─────────────────────┘                   │
└─────────────────────────────────────────────────────────────┘
```

## 4. 현실적 기대효과 (Expected Benefits)

VoiceUp 사용 전/후의 실제적인 변화입니다. 마케팅 자료나 서비스 소개 시 활용하세요.

### 4.1 객관적인 피드백

| 기존 방식 | VoiceUp |
|----------|---------|
| 친구나 가족에게 피드백을 부탁하면 예의상 "괜찮아"라는 말만 듣기 쉬움 | AI는 감정 없이 솔직하게 개선점을 짚어줌. 실제로 뭐가 부족한지 파악 가능 |

### 4.2 내 상황에 맞는 질문 (Context-Based Questions)

| 기존 방식 | VoiceUp |
|----------|---------|
| 인터넷 "면접 예상 질문"은 범용적이라 내 상황과 맞지 않음 | 이력서/발표자료를 분석해 "이 프로젝트에서 가장 어려웠던 점은?" 같은 맞춤 질문 생성 |

### 4.3 컨텍스트 기반 답변 분석

| 기존 방식 | VoiceUp |
|----------|---------|
| 일반 분석은 "구체성이 부족해요"라는 추상적 피드백만 제공 | 내 이력서를 알기에 "OO 프로젝트의 성과 수치를 언급하면 더 설득력 있어요"처럼 맞춤 조언 |

### 4.4 "나도 할 수 있다"는 자신감

| 기존 방식 | VoiceUp |
|----------|---------|
| 다른 사람의 발표를 보면 "저 사람이니까 가능하지"라고 생각하기 쉬움 | 내 목소리로 개선된 버전을 들으면 "나도 이렇게 말할 수 있구나"를 직접 체감 |

### 4.5 막연한 불안 → 구체적 준비

| 기존 방식 | VoiceUp |
|----------|---------|
| 면접 전날, "잘할 수 있을까?" 막연한 걱정만 반복 | "전달력은 좋으니 내용 구체성만 보완하면 돼"처럼 명확한 액션 아이템 제공 |

### 4.6 성장의 연속성

| 기존 방식 | VoiceUp |
|----------|---------|
| 다른 앱들은 매번 처음부터 시작. 어제 받은 피드백을 기억 못 함 | AI가 이전 연습을 기억하고 "저번보다 속도가 좋아졌어요"처럼 성장을 인정 |

### 4.7 요약: VoiceUp의 핵심 가치

```
┌─────────────────────────────────────────────────────────────────┐
│                    VoiceUp 기대효과 요약                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  ✓ 예의 바른 거짓말 대신 → 솔직한 AI 피드백                        │
│  ✓ 범용 예상 질문 대신 → 내 이력서 기반 맞춤 질문                   │
│  ✓ 추상적 피드백 대신 → 내 경험 언급하는 구체적 조언               │
│  ✓ "저 사람이니까" 대신 → "나도 가능하다" 자신감                    │
│  ✓ 막연한 걱정 대신 → 명확한 액션 아이템                           │
│  ✓ 매번 리셋 대신 → 성장을 기억하는 AI                             │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 5. 비주얼 아이덴티티

### 브랜드 네임

**VoiceUp** - "목소리를 높이다" / "목소리로 성장하다"

### 로고 사용 가이드

- 텍스트 로고: "VoiceUp" (Playfair Display)
- 최소 여백: 로고 높이의 50% 이상
- 배경: cream 또는 white 권장

### 컬러 팔레트

| 용도 | 컬러명 | HEX | 사용 |
|------|--------|-----|------|
| **Primary** | Cream | `#FDF8F3` | 배경, 전체 톤 |
| **Accent 1** | Teal | `#0D9488` | CTA 버튼, 강조 |
| **Accent 2** | Coral | `#F97316` | 포인트, 경고 |
| **Text** | Charcoal | `#1C1917` | 본문 텍스트 |
| **Muted** | Stone | `#78716C` | 보조 텍스트 |

### 타이포그래피

| 용도 | 폰트 | 스타일 |
|------|------|--------|
| **본문** | Noto Sans KR | 400, 500, 600 |
| **포인트/타이틀** | Playfair Display | Italic |
| **UI/버튼** | Noto Sans KR | 500, 600 |

### 디자인 원칙

1. **Clean & Professional:** 군더더기 없는 깔끔한 레이아웃
2. **Warm & Approachable:** Cream 톤으로 부드러운 인상
3. **Trust-Building:** 충분한 여백, 명확한 계층 구조
4. **Mobile-First:** 모바일에서도 편안한 사용성

## 6. 브랜드 퍼스낼리티

| 특성 | 설명 | 표현 방식 |
|------|------|----------|
| **전문적** | AI 기술력, 체계적인 분석 | 구체적 수치, 명확한 피드백 |
| **신뢰감** | 데이터 보안, 개인정보 보호 | 투명한 정책 안내 |
| **실용적** | 바로 적용 가능한 개선안 | 즉시 실행 가능한 팁 제공 |
| **지지적** | 성장을 응원하는 파트너 | 성장 포인트 인정, 격려 |

### 브랜드가 아닌 것

- 딱딱하고 권위적인 선생님
- 과도하게 감성적인 친구
- 기술만 자랑하는 스타트업

### 브랜드인 것

- 믿을 수 있는 전문 컨설턴트
- 실력 있는 개인 코치
- 내 성장을 기억하는 파트너
