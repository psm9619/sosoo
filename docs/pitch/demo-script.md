# VoiceUp 데모 시나리오

**Version:** 1.0
**Duration:** 4-5분 (전체 발표 중 핵심 파트)
**Target:** 데모데이 / 경진대회

---

## 사전 준비

### 필수 준비물

| 항목 | 상세 |
|------|------|
| **테스트 계정** | 데모용 계정 (실제 데이터 없는 깨끗한 상태) |
| **이력서 PDF** | "김민지_마케팅매니저_이력서.pdf" (가상 인물) |
| **사전 녹음** | 30초 Voice Clone 샘플 (미리 등록 완료) |
| **백업 영상** | 전체 데모 녹화본 (네트워크 이슈 대비) |
| **마이크** | 테스트 완료 (주변 소음 체크) |

### 환경 설정

- [ ] 브라우저 캐시 클리어
- [ ] 불필요한 탭 모두 닫기
- [ ] 알림 무음 설정
- [ ] 화면 해상도 확인 (1080p 권장)
- [ ] 인터넷 연결 속도 테스트

## Demo 1: Context-Based Practice (1분 30초)

### 목표

> "이력서 업로드 → 맞춤 질문 자동 생성" 과정을 보여준다

### 대본

**[화면: 빈 프로젝트 목록]**

> "먼저 Context 기반 연습을 보여드리겠습니다.
> 기존 앱들은 '자기소개 해보세요', '강점이 뭔가요' 같은 범용 질문만 줍니다.
> VoiceUp은 다릅니다."

**[액션: 새 프로젝트 만들기 클릭]**

> "프로젝트를 만들고, 이력서를 업로드해보겠습니다."

**[액션: 이력서 PDF 드래그앤드롭]**

> "김민지 씨의 마케팅 매니저 이력서입니다."

**[화면: AI 분석 중 로딩]**

> "AI가 이력서를 분석합니다.
> 키워드, 경험, 성과를 추출하고..."

**[화면: 맞춤 질문 14개 생성됨]**

> "보시는 것처럼 14개 맞춤 질문이 생성됐습니다.
>
> - '마케팅 팀에서 어떤 성과를 냈나요?'
> - 'A/B 테스트로 전환율을 높인 경험을 설명해주세요'
>
> 이력서에 있는 내용을 기반으로 면접관이 실제로 물어볼 법한 질문입니다.
> **'어떻게 말할지'뿐 아니라 '무엇을 말할지'까지 준비**할 수 있습니다."

## Demo 2: Analysis + Voice Clone (2분)

### 목표

> "녹음 → AI 분석 → 내 목소리로 개선안 청취" 전체 플로우

### 대본

**[화면: 질문 상세 페이지]**

> "이제 실제로 연습해보겠습니다.
> '마케팅 성과를 설명해주세요' 질문을 선택합니다."

**[액션: 녹음 버튼 클릭]**

> "녹음을 시작하고... 약 30초 정도 답변하겠습니다."

**[액션: 실제 녹음 - 의도적으로 "어", "음" 포함, 빠른 속도로]**

> *"어... 마케팅 팀에서 음... A/B 테스트를 진행했고 전환율이 어...
> 15% 올랐습니다 그래서 음... 성과가 좋았습니다."*

**[액션: 녹음 중지, 분석 시작]**

> "녹음을 마치고 AI 분석을 시작합니다."

**[화면: 분석 중 로딩 + Quote 표시]**

> "분석하는 동안 영감을 주는 문구를 보여드립니다."

**[화면: 분석 결과]**

> "분석 결과를 보시면...
>
> - 속도: 185 WPM, 권장보다 빠릅니다 → C
> - 필러워드 6% → C+
> - 구조: STAR 중 Task가 부족합니다 → B
>
> 그리고 **구체적인 개선 포인트 3가지**를 알려줍니다."

**[액션: 개선안 재생 버튼 클릭]**

> "여기서 핵심입니다.
> 개선안을 **제 목소리**로 들어보겠습니다."

**[화면: 개선안 재생 - Voice Clone 음성]**

> *[Voice Clone 음성이 재생됨:
> "마케팅 팀에서 전환율 개선 프로젝트를 담당했습니다.
> 목표는 3주 내 10% 향상이었고,
> 5가지 A/B 테스트를 설계하고 실행했습니다.
> 결과적으로 15% 전환율 상승을 달성했습니다."]*

> "들으셨나요? 제 목소리입니다.
> 남의 완벽한 발음이 아니라, **'나도 이렇게 말할 수 있구나'**를 직접 체감합니다.
>
> 이게 VoiceUp의 핵심 차별점입니다.
> **분석만 하는 게 아니라, 개선안을 내 목소리로 직접 시연**합니다."

## Demo 3: Progressive Memory (1분)

### 목표

> "2회차 분석에서 성장 패턴 인식" 보여주기

### 대본

**[화면: 히스토리에서 2회차 결과 선택 (미리 준비)]**

> "마지막으로, Progressive Memory를 보여드리겠습니다.
> 이건 미리 연습한 2회차 결과입니다."

**[화면: 2회차 분석 결과]**

> "보시면 AI가 **이전 세션과 비교**해서 피드백합니다.
>
> 📈 **성장 포인트:**
> '지난 세션 대비 속도가 185에서 160으로 개선되었습니다.
> 의식적으로 노력하신 게 느껴지네요.'
>
> ⚠️ **아직 개선 중:**
> '필러워드가 여전히 5%입니다.
> 문장 끝에서 잠깐 멈추는 연습을 계속해보세요.'
>
> 다른 서비스들은 매번 독립적으로 분석합니다.
> VoiceUp은 **당신의 성장을 기억하고, 지속적으로 코칭**합니다."

## Demo 마무리 (30초)

> "정리하면,
>
> 1. **내 이력서로 맞춤 질문 생성** - 무엇을 말할지
> 2. **AI 분석 + 내 목소리로 시연** - 어떻게 말할지
> 3. **성장을 기억하는 AI** - 지속적 개인화
>
> 이 세 가지가 VoiceUp의 핵심 차별점입니다.
>
> **'분석 + 시연 + 내 목소리' 삼박자를 모두 갖춘 서비스는 VoiceUp이 유일합니다.**"
