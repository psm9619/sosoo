# VoiceUp One-Pager

**AI 기반 스피치 코칭 SaaS**

---

## Problem

> "면접/발표 연습하는데 뭐가 문제인지 모르겠어요"

- 혼자 연습해도 객관적 피드백 받기 어려움
- 전문 코칭은 $100+/시간, 시간/장소 제약
- 기존 AI 앱은 점수만 주고 "어떻게 고치라는 건지" 없음

---

## Solution

### VoiceUp - "내 상황, 내 목소리로 완성하는 AI 스피치 코칭"

| 차별점 | 설명 | 경쟁사 대비 |
|--------|------|------------|
| **Voice Cloning** | 개선안을 내 목소리로 시연 | Orai, Yoodli에 없음 |
| **Context Practice** | 이력서/자료 분석 → 맞춤 질문 생성 | 범용 질문만 제공 |
| **Progressive Memory** | AI가 성장 패턴 학습 → 개인화 피드백 | 세션별 독립 분석만 |

---

## How It Works

```
1. 프로젝트 생성 → 이력서/발표자료 업로드 (선택)
2. AI가 맞춤 질문 생성
3. 질문에 답변하며 녹음
4. AI 분석 → 속도, 필러워드, 구조, 내용 피드백
5. 개선안을 "내 목소리"로 청취
6. Before/After 비교 → 반복 연습
```
