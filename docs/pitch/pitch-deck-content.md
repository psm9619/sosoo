## 슬라이드 3: Problem

### 객관적 피드백의 부재

| 기존 방법 | 문제점 |
|----------|--------|
| **혼자 연습** | 뭐가 문제인지 스스로 판단 불가 |
| **친구에게 부탁** | 구체적 피드백 어려움, 부담 |
| **전문 코치** | $100+/시간, 시간/장소 제약 |
| **기존 AI 앱** | 점수만 주고, "어떻게 고치라는 건지" 없음 |

**핵심 Gap:** 구체적 개선안을 **직접 보여주는** 서비스가 없다

---

## 슬라이드 4: Solution

### VoiceUp - 3대 핵심 차별점

| # | 차별점 | 설명 | 경쟁사 대비 |
|---|--------|------|------------|
| 1 | **Voice Cloning** | 개선안을 **내 목소리**로 시연 | Orai, Yoodli에 없음 |
| 2 | **Context Practice** | 이력서/발표자료 → 맞춤 질문 생성 | 대부분 범용 질문만 |
| 3 | **Progressive Memory** | AI가 성장 패턴 학습 → 개인화 피드백 | 세션별 독립 분석만 |

> **"분석 + 시연 + 내 목소리"** 삼박자를 모두 갖춘 유일한 서비스

## 슬라이드 6: Demo 2 - Analysis + Voice Clone

### "녹음하면 AI가 분석하고, 내 목소리로 개선안을 들려준다"

**[LIVE DEMO]**

1. 질문 선택 → 녹음 (1분)
2. AI 분석 결과:
   ```
   📊 스피치 분석
   ├─ 속도: 185 WPM (권장: 120-170) → C
   ├─ 필러워드: 6% (목표: <3%) → C+
   ├─ 구조 (STAR): B+
   └─ 내용 명확성: B

   💡 개선 포인트:
   1. 결론을 먼저 말하세요
   2. 속도를 10% 낮추세요
   3. "어, 음" 대신 짧게 멈추세요
   ```
3. **개선안 재생** → 내 목소리로 듣기
4. Before/After 비교 청취

**Point:** 남의 완벽한 발음이 아닌, **"나도 이렇게 말할 수 있구나"** 체감

## 슬라이드 8: Technology

### AI Pipeline Architecture

```
┌────────────────────────────────────────────────────────────┐
│                    VoiceUp AI Pipeline                      │
│                                                             │
│  Audio → STT(Whisper) → Analysis(Claude) → TTS(ElevenLabs)  │
│            │                  │                    │        │
│            │                  │                    │        │
│      음성→텍스트        전문 코치 분석        내 목소리 생성  │
└────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│                  Progressive Context                         │
│                                                              │
│  Long-term Memory          +        Short-term Memory        │
│  (이력서/자료 분석)                  (최근 3세션 패턴)        │
│          │                               │                   │
│          └───────────┬───────────────────┘                   │
│                      ▼                                       │
│           개인화된 맞춤 피드백                                 │
└─────────────────────────────────────────────────────────────┘
```

### Tech Stack

| Component | Technology | 역할 |
|-----------|------------|------|
| **STT** | OpenAI Whisper | 음성 → 텍스트 |
| **Analysis** | Claude Sonnet | 전문 코치 수준 분석 |
| **TTS** | ElevenLabs | Voice Cloning |
| **Workflow** | LangGraph | AI 파이프라인 오케스트레이션 |
| **Frontend** | Next.js 15 + React 19 | 반응형 웹앱 |
| **Backend** | Supabase | Auth, DB, Storage |

## 슬라이드 10: Competition

### 경쟁 환경

| Player | 강점 | Gap |
|--------|------|-----|
| **전문 코치** | 고품질 피드백 | $100+/시간, 접근성 |
| **Orai** | 글로벌 1위 앱 | Voice Clone 없음 |
| **Speeko** | 게이미피케이션 | 개선안 시연 없음 |
| **Yoodli** | AI 면접 분석 | Context 기반 없음 |

### VoiceUp의 진입장벽

1. **Voice Cloning**: 단순 TTS가 아닌 "내 목소리" 시연
2. **Progressive Context**: 축적된 분석 데이터 → 개인화 정확도
3. **한국어 최적화**: 한국어 필러워드, STAR 구조 분석
